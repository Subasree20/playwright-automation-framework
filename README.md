# Playwright Automation Framework

This project demonstrates an end-to-end UI automation framework built using Playwright with JavaScript, following best practices such as Page Object Model (POM) and reusable test design. It includes an end-to-end booking flow with validation before and after booking.

---

## 🚀 Features

- End-to-End automation for real-time scenarios (Event Booking Flow)
- Page Object Model (POM) implemented for maintainability
- Reusable utilities and modular structure
- API validation support
- Assertions and validations for UI testing

---

## 🛠️ Tech Stack

- JavaScript (Node.js)

## ▶️ Run Tests

Install dependencies:
```
npm install
```

Run tests:
```
npx playwright test
```

Run in headed mode:
```
npx playwright test --headed
```
